PHOTO DE PROFIL
===============

Déposez votre photo ici sous le nom exact : yann.jpg

Chemin final attendu :
  portfolio/public/yann.jpg

Formats supportés : .jpg, .jpeg, .png, .webp
(Si vous changez l'extension, mettez à jour src dans components/Avatar.tsx)

Format recommandé :
- Carré (1:1)
- Minimum 400x400 px
- Max 2 Mo

Sans photo, l'avatar affichera un fallback "YT" sur fond dégradé.
